# Проект: Обучение с учителем — Прогноз снижения активности клиентов

## Постановка задачи
Интернет-магазин «В один клик» столкнулся со снижением активности постоянных клиентов. Цель проекта — построить модель машинного обучения для прогнозирования вероятности снижения покупательской активности в ближайшие 3 месяца и на основе этого выделить приоритетные сегменты для удержания.

## Используемые модели
- Logistic Regression (с StandardScaler и OneHotEncoder)
- Support Vector Classifier (SVC)
- K-Nearest Neighbors (KNN)
- Decision Tree

Все модели обучены в пайплайнах с предобработкой признаков и подбором гиперпараметров через `GridSearchCV`.

## Результаты
- Лучшая модель: **Logistic Regression**
- F1-score на тестовой выборке: **0.857**
- Ключевые признаки, влияющие на отток:  
  - Низкое количество страниц за визит  
  - Высокая доля акционных покупок  
  - Большое число неоплаченных товаров  
  - Низкая маркетинговая активность

На основе прогнозов выделен сегмент **«Высокая прибыльность + Высокий риск»** (108 клиентов, средняя прибыль — 518 у.е.), для которого предложены персонализированные маркетинговые стратегии.
